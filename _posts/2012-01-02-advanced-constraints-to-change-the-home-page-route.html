--- layout: post title: Advanced constraints to change the home page route tags: - programming - Programming - rails status: publish type: post published: true comments: true meta: dsq_thread_id: '524744070' _edit_last: '1' --- I just finished implementing the excellent guide by Steve Richert (http://collectiveidea.com/blog/archives/2011/05/31/user-centric-routing-in-rails-3/) to change the home page route based on whether the user is logged in or not. The last comment on the thread gave me the solution for using with sessions instead of cookies: {% highlight ruby %} request.session[:your_key] {% endhighlight %} instead of {% highlight ruby %} request.cookies.key?("user_token") {% endhighlight %} I need to implement cookies next and make logins rememberable. Pretty excited about this first little programming task of the year.
