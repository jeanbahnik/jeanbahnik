--- layout: post title: Mailer layout in rails 3.1 tags: - Programming - rails status: publish type: post published: true meta: _edit_last: '1' dsq_thread_id: '491224130' --- You can use a layout for your emails just like the layout file for your website. I just implemented it to keep the look of utellmewhen's emails clean and consistant. Here is the link to the Ruby Guide on it: http://guides.rubyonrails.org/action_mailer_basics.html#action-mailer-layouts. I first created my email in html, sticking with tables to format the email over divs. I included the css styling into the elements, not in a header. Then I saved the file as user_mailer.html.erb in the views/layouts folder. I initially saved it in my views/user_mailer by mistake. Make sure you put the layout... in the layouts folder. Then I put <%= yield %> in my layout file and that was it. You can also override the layout that you use with: 
class UserMailer < ActionMailer::Base
  layout 'awesome' # use awesome.(html|text).erb as the layout
end
Hope this helps.
