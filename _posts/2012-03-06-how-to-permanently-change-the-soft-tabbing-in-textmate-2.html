From the terminal run, assuming TextMate is installed in the Applications folder: {% highlight shell-session %} mate ~/../../Applications/TextMate.app/Contents/Resources/Default.tmProperties {% endhighlight %} Then add the following lines: {% highlight python %} [ "*.{haml,yml,coffee,feature}" ] softTabs = true tabSize = 2 {% endhighlight %}
