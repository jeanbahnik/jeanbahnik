--- layout: post title: Using rak to search for code or file tags: - Programming - rak - ruby - terminal status: publish type: post published: true meta: _edit_last: '1' dsq_thread_id: '720542488' --- Today Sarah Grey introduced me to rak during our pairing session. rak makes it very easy to search for a piece of code in your project, or search for files by their name.
From http://rak.rubyforge.org/:
"Rak is a grep replacement in pure Ruby. It accepts Ruby syntax regular expressions and automatically recurses directories, skipping .svn/, .cvs/, pkg/ and more things you don't care about."
After installing the gem through: 
gem install rak
just type 
rak search_term
to find the search_term in your code. There are a lot of options such as [FILES] to search on those files/directories,  -i to ignore the case or --group to group matches by file name. Go check it out at http://rak.rubyforge.org/.
I just used it to find examples of inject. Let me know if it helps you.
